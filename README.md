Pagina unica de construcao civil
Este trabalho e uma landing page (uma pagina unica) que fiz para treinar. Foi construida apenas com html e css puro. Os botoes e os links tem efeitos visuais quando se passa o rato, mas sao apenas de demonstracao e nao levam a outras paginas.
Nota sobre o visual: o visual deste projeto foi copiado e inspirado num template que encontrei online. Serviu estritamente como exercicio de estudo para eu treinar como passar um desenho para codigo html e css.

-como ver o site:

 1 faz download do ficheiro zip no botao verde
 2 extrai a pasta
 3 abre o ficheiro html no teu navegador
